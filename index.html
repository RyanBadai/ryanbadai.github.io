Clothes Recognition with CNN Method in My Capstone Project at Bangkit Program 2023

In the increasingly advanced digital era, recognition and classification of objects in images has ... become an interesting and important research topic. One application that attracts attention in this regard is shirt type recognition. Shirt type recognition has many potential applications, such as in the fashion industry, e-commerce, and the development of interactive systems that require an understanding of the types and characteristics of clothes. In this journal, the author uses the Convolutional Neural Network (CNN) method. CNN or Convolutional Neural Network is one of the Machine Learning methods from the development of Multi Layer Perceptron (MLP) which is designed to process two-dimensional data. CNN is a type of Deep Neural Network because it includes a network level and is widely implemented in image data. After the model is run with the number of iterations or epochs of 100 epochs, the Accuracy result is 90.4% and Model Loss is 0.29%. Based on the test results, it can be concluded that using MobileNet architecture can get better results.
